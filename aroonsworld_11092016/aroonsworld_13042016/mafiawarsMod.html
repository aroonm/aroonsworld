Mafia Wars

Mafia Wars is a crime-genre, social multi-player game developed by Zynga. It started out with an initial player base of 45 million users on Facebook. As a full-stack software developer at Zynga, I contributed to this game in several ways:

  • Writing effiecient, elegant backend code in PhP.
  • Front-end development using HTML, CSS, JavaScript, Ajax and Jquery.
  • Developing features foucussing on player-retention and monetization.
  • Writing Automation Tools on my own time to reduce time spent by Developers and QA on cadences/content refreshes. Working on a HelpShift Module to increase efficiency with which are CS team worked.
  • Working with a multi-disciplinary team consisting of Producers, Game Designers, Artists, UI/UX and Narrative Designers.
  • Using Agile Methodologies for quick product development and delivery.
  • Winning the "Emerging Star" award.

I'm really proud to have been a part of an amazing team and an iconic game.
